# Job Portal

A full-stack MERN Job Portal that connects students with recruiters through a modern recruitment platform.

## Features

### Student

* User Authentication (Signup/Login)
* Profile Management
* Resume Upload
* Browse Jobs
* Search Jobs by Title, Company and Location
* Apply for Jobs
* View Applied Jobs

### Recruiter

* Company Registration and Management
* Company Logo Upload
* Create and Manage Job Postings
* View Applicants

## Tech Stack

* React.js
* Redux Toolkit
* Node.js
* Express.js
* MongoDB
* Cloudinary
* Tailwind CSS

## Key Enhancements

* Improved authentication flow with automatic login after registration
* Enhanced search functionality supporting company name search
* Resume upload integration using Cloudinary
* Improved recruiter dashboard and company management workflow
* Modernized UI with responsive design

## Installation

```bash
npm install
npm run dev
```
